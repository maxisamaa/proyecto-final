# Gestor Personal de Tareas, Pensamientos y Referencias

Este proyecto es una aplicación web desarrollada con Django que integra diversas aplicaciones internas —tareas, categorías, etiquetas, referencias y usuarios— para ofrecer una herramienta completa orientada a organizar planes, prioridades, pensamientos y referencias de lecturas dentro de un entorno privado y personalizado.

## Características Principales

La aplicación permite crear tareas que pueden filtrarse mediante etiquetas temáticas como **cotidianidad**, **filosofía**, **conocimiento**, **arte**, entre otras, y cada tarea incluye tres tipos de clasificación —**proximidad**, **prioridad** y **tipo**— que permiten ordenar las actividades de la forma que resulte más útil para el usuario; todo el sistema está protegido con la autenticación de Django, por lo que cada usuario únicamente puede visualizar sus propias tareas y no tiene acceso al contenido de otros, garantizando completa privacidad; la sección de referencias ofrece dos tipos de registros —**pensamientos** y **referencias de lectura**— cada uno con formularios y estructuras de almacenamiento independientes para mantener claridad y orden; adicionalmente, el sitio incluye un gadget visual que muestra una grilla con todas las semanas de una vida promedio, indicando cuántas semanas has vivido y cuántas quedarían, funcionando como un recurso ilustrativo para reflexionar sobre prioridades y organización temporal.

## Uso General

Para utilizar la plataforma solo debes acceder a la URL del sitio web, registrarte o iniciar sesión y desde el panel principal podrás crear tareas asignando etiquetas y categorías, revisar o ingresar pensamientos y referencias según corresponda, y utilizar el sistema de filtrado por etiquetas para visualizar solo lo necesario en cada momento; todo el contenido permanece siempre aislado y protegido por la autenticación incorporada, ofreciendo un espacio personal y seguro para la organización tanto práctica como intelectual.
